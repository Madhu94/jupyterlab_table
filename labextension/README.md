# jupyterlab_table JupyterLab extension

A JupyterLab extension for rendering JSON Table Schema

## Prerequisites

* `jupyterlab@^0.17.0`

## Development

Install dependencies and build Javascript:

```bash
npm install
```

Re-build Javascript:

```bash
npm run build
```

Watch `/src` directory and re-build on changes:

```bash
npm run watch
```
